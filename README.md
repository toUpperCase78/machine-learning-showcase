# My Machine Learning Showcase

This repository is my showcase about the research of many machine learning models (both classification & regression) for the purpose of yearly salary predictions. This also includes additional works such as applied feature engineering on the dataset, metric measurements, Natural Language Processing (NLP), topic modeling with LDA and explainable AI (SHAP / LIME).
